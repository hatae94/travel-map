# 풀스택 바이브코딩 스터디 - 보일러플레이트

## 구조

```
├── backend/    # NestJS v9 (pg 라이브러리 설치됨)
├── mobile/     # React Native (Expo SDK 55)
```

## 시작하기

### 1. 백엔드 실행

```bash
cd backend
yarn install
yarn start:dev
```

### 2. 모바일 실행

```bash
cd mobile
yarn install
yarn prebuild
yarn ios
yarn android
```
